# 文字コードあれこれ
## Unicodeコードポイントと文字の変換をめぐって
シェル芸をやる都合、コードポイントと文字の対応をよく参照するのでそのための備忘録…φ

1. 文字からコードポイント(16進)を出力する方法
```ruby
puts "あ".ord.to_s(16)
=> 3042
```

2. コードポイント(16進)から文字を出力する方法
```ruby
puts "0x3042".hex.chr("UTF-8")
=> あ
```
